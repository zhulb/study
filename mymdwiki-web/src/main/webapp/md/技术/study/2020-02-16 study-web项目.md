# 学习、积累、悟道应该要有一个模式

重要性：就像设计模式对程序的重要性一样，学习的过程也需要规律高效的模式。

方式：而模式的设计则需要根据自己学习的根本需求、目的去设计，如有些在生活中重要的经验需要了解与应用；而有些重要的启示、智慧则需要牢记于心，
所以必须先抓取出学习中的实际需求、需要，然后进行取舍，最后在根据这些需求去设计自己的学习模式。
